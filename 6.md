#### HQ-RTR:
Добавляем туннель

**mcedit /etc/network/interfaces**

```
auto gre1
	iface gre1 inet tunnel
	address 10.10.10.1
	netmask 255.255.255.252
	mode gre
	local 172.16.4.2
	endpoint 172.16.5.2
	ttl 255
```
Применяем

**systemctl restart networking**
